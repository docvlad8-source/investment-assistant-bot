Основные формулы:

1. YTM (доходность к погашению):
   Решается численно: Цена = Σ(Купон / (1+YTM)^t) + Номинал / (1+YTM)^T

2. Дюрация Маколея:
   D = Σ(t × PV(CF_t)) / Цена

3. P/E = Цена акции / EPS

4. Дивидендная доходность = Дивиденд / Цена

5. CAPM: r = r_f + β(r_m - r_f)

6. WACC = (E/V)×r_e + (D/V)×r_d×(1-T)

7. Sharpe Ratio = (r_p - r_f) / σ_p

8. NPV = Σ(CF_t / (1 + r)^t)

9. IRR: NPV = 0 → найти r

10. Текущая доходность облигации = Купон / Цена
